# Windows compatability
- screenshot
- close window, app
- emoji
- alt-tab
- fullscreen

## Heading
 * maximize/minimize, move to (left/right) requires going into mouse mode first.
* neovim, zen browser, and tmux should be configured to match
  - neovim (close should be <C-w>)
  - tmux (switch panes is Alt-a)

- I lose the ability to toggle between monitors
  - use a wider monitor and position items side by side (up to 3 windows)

- use super tab to view the task viewer and then click the + icon
- to simulate the scratchpads, create 2 virtural desktops manually and then use the scratchpad keybinds to switch between the 2 virtual desktops.


# Mac compatability
- I don't think I will add support for a mac. It requires too many modifications

# Symbols
- 32 symbols altogether
  - Enclosers: [] () {} " ' ` (9)
  - Math and Equality: /*=+- %#$^ <> (11)
  - Connectors/Punctuation: _ . , ; : !? (7)
  - Other: \&|~@ (5)

## Text Layer
- Base Symbols: '-.,-*
- Base Combo Symbols: _;:/!?
  - These are also available in `Symbol` layer

## Numpad Layer
- TR: top right (math based symbols)
  - $=+

## Symbols Layer
- Top: &<>\-  *$=+f
- Mid: _;:/R  R[({"
- Bot: OSL.BS,AREP  s])}OSL

## Flow Symbols
> purpose of it is to make harder to reach keys more accessible while within the Symbols layer
- Top: t%$tt ttttt
- Mid: '-*/t t#|`%
- Bot: ta!t?t t^~@t
